# recordmydesktop 2.0 - r2d2.sh

Usage: r2d2.sh [options] [output filename].webm

Options:

	-h this help
	-v verbose
	-d duration in seconds
	-u upload host

Example:

	r2d2.sh -d 5 -u r2d2.webconverger.org:/srv/www/r2d2.webconverger.org

Overwrites to `out.webm` by default.
